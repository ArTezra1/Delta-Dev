O design é uma disciplina ampla e multifacetada que envolve a criação e organização de elementos visuais para comunicar mensagens, conceitos ou ideias de forma eficaz e esteticamente agradável.

Um sistema web é uma aplicação de software projetada para ser acessada e utilizada através de um navegador da web. Em resumo, ele opera na arquitetura cliente-servidor, onde o cliente (navegador) faz solicitações para um servidor web, que processa essas solicitações e fornece uma resposta de volta ao cliente.
